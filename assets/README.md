# Assets Folder

Place your app icons and images here:

- `icon.png` - App icon (1024x1024)
- `splash-icon.png` - Splash screen icon (512x512)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

## Generating Icons

You can use tools like:
- https://makeappicon.com
- https://appicon.co
- Figma with App Icon plugin

## Recommended Icon Design

For "Give Me Guidance", consider:
- A dove or cross symbol
- A book/bible icon
- Green gradient background (#10b981 to #14b8a6)
- Clean, modern design
